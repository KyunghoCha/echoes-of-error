## 2. File Naming Convention

- **Markdown:** `Title_Case_With_Underscores.md` (e.g., `Data_Analysis_Plan.md`)
- **Scripts:** `snake_case.py` (e.g., `run_batch.py`)
- **Logs:** `YYYYMMDD_HHMMSS_Scenario_Mode.jsonl`

## 3. Maintenance Rules

- **No Fragmentation:** Do not create new top-level folders in `docs/` without a strong reason. Use the existing categories.
- **Archive Old Plans:** When a plan is obsolete, move it to `docs/archive/` or mark it as `[DEPRECATED]` in the header.
